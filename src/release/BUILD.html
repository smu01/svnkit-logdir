SVNKit LogDir protocol implementation

Version: 0.1.2

Build

  1. Get the source code from GitHub
  2. Compile the source code with Maven2: 
    mvn package

Use

  1. Add svnkit-1.2.x.jar and svnkit-logdir-a.b.c.jar to your project’s build path
  2. Register the repository factory by calling 
    LogDirSVNRepositoryFactory.setup();
  3. Use the normal SVNKit API (Subversion URLs need to have the format logdir:///path/to/logs)
